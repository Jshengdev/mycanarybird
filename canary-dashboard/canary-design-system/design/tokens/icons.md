# Canary Design System — Icons
> Parent: `design/INDEX.md`

---

## Library
Lucide Icons — stroke-based, 1.5px stroke weight.
Import: `import { IconName } from 'lucide-react'`

## Sizes
| Size | Usage |
|---|---|
| 12px | Inline/compact — badges, table cells, sidebar sub-items, button icons |
| 16px | Standard — standalone icon buttons, nav items |

## Color
Default: `--icon-grey (#858585)`
Active/selected: `--text-black (#121212)`
Error context: `--critical (#FF2E2E)`
Success context: `--safe (#48C72B)`
